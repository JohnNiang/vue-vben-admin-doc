# 快速上手

前提条件

Vue-Vben-Admin (opens new window)需要 Node.js (opens new window)版本>=12

本文会帮助你从头启动项目。

# 工具配置

如果您使用的 IDE 是vscode (opens new window)的话,可以安装以下工具来提高开发效率就代码格式化

  • Iconify IntelliSense (opens new window) - Iconify 图标插件
  • Vetur (opens new window) - vue 开发必备
  • ESLint (opens new window) - 脚本代码检查
  • Prettier (opens new window) - 代码格式化
  • Stylelint (opens new window) - css 格式化
  • DotNev (opens new window) - .env 文件 高亮

# 代码获取

TIP

注意存放代码的目录及所有父级目录不能存在中文、韩文、日文以及空格。否则安装依赖后启动会出错。

# 从 github 获取代码

# clone 代码
git clone https://github.com/anncwb/vue-vben-admin

2
3

# 从 gitee 获取代码

如果从 github clone 代码较慢的话，可以尝试用 gitee 同步代码到自己的仓库，在 clone 下来即可。

也可以通过下方地址进行 clone

git clone https://gitee.com/annsion/vue-vben-admin-next
1

注意

vue-vben-admin-next (opens new window)在 gitee 的代码可能不是最新的

# 安装

# 安装 Node.js

如果您电脑未安装Node.js (opens new window)，请安装它。

验证

# 出现相应npm版本即可
npm -v

# 出现相应node版本即可
node -v

2
3
4
5
6

如果你需要同时存在多个 node 版本，可以使用Nvm (opens new window)或者其他工具进行 Node.js 进行版本管理。

# 安装依赖

yarn 安装

推荐使用 Yarn (opens new window)进行依赖安装。

如果未安装yarn，可以用下面命令来进行全局安装

# 全局安装yarn
npm i -g yarn

# 验证
yarn -v # 出现对应版本号即代表安装成功
1
2
3
4
5

其他包管理工具类似

依赖安装

在项目根目录下,打开命令窗口执行,耐心等待安装完成即可

# 安装依赖
yarn install

# npm Script

安装依赖

yarn bootstrap
1

运行项目

yarn serve
1

打包项目

yarn build
1

打包项目部署站点

用于 Vue Vben Admni 打包部署预览使用

yarn build:site
1

清空缓存后构建

yarn build:no-cache
1

生成打包分析

在Mac电脑上执行完成后会自动打开界面

在Window电脑上执行完成后需要打开./build/.cache/stats.html查看

yarn report
1

预览打包后的内容

# 先打包在进行预览
yarn preview

# 直接预览本地 dist 文件目录
yarn preview:dist
1
2
3
4
5

生成 ChangeLog

yarn log
1

删除缓存

yarn clean:cache
1

删除 node_modules

window系统手动删除该目录较慢，可以使用该命令来进行删除

yarn clean:lib
1

重新安装依赖

该命令会先删除 node_modules、yarn.lock、package.lock.json后在进行依赖重新安装

速度会明显变慢

yarn reinstall
1
